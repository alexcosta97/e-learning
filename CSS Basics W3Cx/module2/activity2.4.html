Main Title

In unit 2.3, we defined CSS selectors as the portion of the CSS rule that tells the browser on which HTML element to apply the defined style. When our HTML is simple, the selectors can be simple as well. The most basic selectors simply mirror the HTML tag. For example, "p" attaches to all < p> tags, "img" will attach to all <image> tags and so on. As we can imagine, there will often be times when we don't want every single HTML element of a particular tag to have identical style.
